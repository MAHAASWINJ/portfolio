~/projects/01

Online Food Delivery Management System

A full-stack DBMS project — from a normalized six-table schema to a working ordering flow.

Built end-to-end with Flask, MySQL, and React, this project models a real food delivery platform: restaurants, menus, users, orders, and reviews. The database is normalized to 3NF with enforced foreign keys, CHECK constraints on price, quantity, and rating, and CASCADE/RESTRICT rules chosen deliberately for each relationship rather than left at defaults.

Flask MySQL / MariaDB React Tailwind CSS Axios Vite mysql-connector-python
  • Restaurant listing with search & per-restaurant categorized menus
  • Cart system with quantity management and full checkout validation
  • Order placement, status tracking, and full order history
  • Star-rating review system tied to verified users
  • Referential integrity enforced at the database level, not just the app
